Welcome to Smartmap

Smartmap is an interactive public transport map application. On this map you can access transport lines and get their directions, actualities on start/stop times and all possible stops

For this project we've created an interactive user interface and a backend application that handles the data. Our api utilizes the data from ovapi

Possible Routes Description
/journey/eindhoven This will return a list of possible journeykeys with data owner code CXX. The region code for Eindhoven.
/actuals/eindhoven/:journeykey This will return a list of the actual information for that particular bus. You will need the journey key from the previous routes to access this data
/lines This will return a list of driving bussess throughout the entire country
/lines/dacs/ This will return a list of area owner codes. These are needed to access data per region. ARR is arriva and CXX is Bravo
/route This will return a list of all coordinates for each line that represent a bus stop. We use these coordinates to build the directions line from start destination to end destination
/route/:line This will return a list of all coordinates for a specific line that represent a bus stop. We use these coordinates to build the directions line from start destination to end destination.
NOTE: You will need to add a capital L before the linenumber otherwise you won't get any data back. possible lines: [L400, L401, L402, L403, L404, L405, L406, L407]

How it works

When the application is launched, a map will be shown along side with an user interface of the possible lines you can select. In this example we've chosen to show how our application builds the directions line and provides all possible bus stops on the map. In the end we want to provide the data per stop when you click on one of the stops.
